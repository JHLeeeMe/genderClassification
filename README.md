
# imgClassification
남녀 얼굴사진을 학습시켜 구분 (tensorflow)
<br>
ImgScrapy.py => data 수집  
FaceDetection.py => 수집된 데이터 얼굴부분만 crop, resize (feat. openCV)  

## requirements
    cv2
    tensorflow
    numpy
    pandas
    urllib
    bs4
    selenium
    
## how to run
    python3 ImgScrapy.py searchWord_1 [searchWord_2] ... [searchWord_N]
    python3 FaceDetection.py
